# PolyKern

This proposal presents a new lookup that implements a polyline outline based kern. The primary purpose for this lookup is to support kerning in situations of complex layout, in particular for Nastaliq Arabic fonts. 

See the [PolyKern Lookup file](https://github.com/silnrsi/polykern/blob/main/documentation/PolyKernLookup.md) in the documentation directory for a description of the proposed new lookup.

**Discussions** have been enabled in this repository. Feedback and discussions for this proposed new lookup should be done in the Discussions area.
